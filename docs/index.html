Generisk skrapmotor

En konfigurationsdriven pipeline som skrapar valfri webbsajt, lagrar historik, exporterar Excel/PDF-rapporter och skickar Discord-notiser om nya träffar — helt automatiserat via ett gratis GitHub Actions cron-jobb. Den här sidan visar upp live-data från en av de konfigurerade sajterna (en filmkatalog).

Fördelning

Ingen data hittad ännu. Kör python scraper.py step3 sites/<namn>.json för att generera docs/data/.

Arkitektur

Samma pipeline körs för alla konfigurerade sajter, oberoende av vilken hemsida som skrapas.

Lägg till en ny sajt genom att skriva en JSON-fil i sites/ — oftast räcker det, ingen kod behöver skrivas (extractor: "generic" läser av titel, meta-beskrivning och "Label: värde"-mönster automatiskt).
